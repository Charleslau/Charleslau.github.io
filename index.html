📱 后台运行指南
iOS 用户（iPhone/iPad）：
1. 设置 → 显示与亮度 → 自动锁定 → 永不
2. 设置 → 辅助功能 → 引导式访问 → 开启
3. 运行应用后三连击电源键开启引导式访问
4. 保持充电，防止低电量自动锁屏
Android 用户：
1. 设置 → 电池 → 应用电池管理 → 允许后台活动
2. 最近任务 → 长按应用 → 锁定
3. 设置 → 应用 → 自启动管理 → 允许自启动
